Get Started
01 What is Reyn

Reyn is an AI agent orchestration system. It gives you structured control over multiple agents through defined flows and phases.

02 How it works

Reyn connects agents through a skill graph.

Each phase is driven by LLM decisions within a constrained, auditable runtime.

03 Key concepts
Agent

One worker, one skill.

A scoped agent with a single skill, a memory window, and the right to call tools.

Flow

A graph of phases.

A directed graph that routes work between agents based on decisions and state.

Control

An auditable runtime.

Every step is logged, replayable, and can be paused or steered mid-flight.
