# Stdlib

Reyn に付属するバンドルされたコンポーネント。

| コンポーネント | 目的 |
|--------------|---------|
| [direct_llm](direct_llm.md) | プロンプトを LLM に直接渡し、生のレスポンスを返す |
| [eval](eval.md) | LLM-as-judge を使って 1 つのテストケースを評価する |
| [eval_builder](eval_builder.md) | eval スペック（`eval.md`）を生成する |
