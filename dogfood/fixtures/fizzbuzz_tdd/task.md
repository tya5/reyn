# Task

The directory contains:

- `fizzbuzz.py` — an unfinished module with a `fizzbuzz(n)` stub that raises `NotImplementedError`.
- `test_fizzbuzz.py` — pytest tests that currently fail because of the stub.

## Your job

1. Read `test_fizzbuzz.py` to understand the required behaviour.
2. Edit `fizzbuzz.py` so all tests pass.
3. Run `python -m pytest test_fizzbuzz.py -q` to verify.
4. If any test fails, read the failure message, fix the implementation, and re-run pytest.
5. Repeat until all tests pass.

When all tests pass, reply with a one-line summary like "all 5 tests pass" and stop.
